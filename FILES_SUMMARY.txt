  3. chmod +x setup-menu.sh
  4. ./setup-menu.sh
  5. Follow the interactive menu
  6. Done! Search "Kenae Media Player" in app launcher

FOR USERS WITH APPIMAGE:
  1. Download AppImage from GitHub Releases
  2. cd media_keyframe (or extract repo)
  3. chmod +x quick-install.sh
  4. ./quick-install.sh
  5. Choose installation type
  6. Done! Search "Kenae Media Player" in app launcher

  1. Clone repository
  2. Make changes to code
  3. ./build-appimage.sh
  4. ./kenae_media_player-x86_64.AppImage (to test)
  5. Repeat steps 2-4 as needed

FOR MAINTAINERS (RELEASES):
  1. Make final changes & test
  2. git tag v1.0.0
  3. git push origin v1.0.0
  4. GitHub Actions automatically builds & releases
  5. AppImage available in GitHub Releases
